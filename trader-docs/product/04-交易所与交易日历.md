# 模块：交易所与交易日历

## 目标与范围

提供交易所基础信息查询与交易日历（开市/休市）查询能力，服务于管理端浏览与其他模块的交易日判断。

## 前端设计（admin-web）

### 页面入口

- 交易所列表：/exchange/list
  - 页面：[List.vue](../../trader-admin/admin-web/src/pages/exchange/List.vue)
- 交易日历：/exchange/calendar
  - 页面：[CalendarList.vue](../../trader-admin/admin-web/src/pages/exchange/CalendarList.vue)

### 关键交互

- 交易所列表：关键字查询 + 分页
- 交易日历：按交易所 code + 日期范围查询 + 分页

## 后端设计（admin-server）

### 交易所

- Controller：[ExchangeController.java](../../trader-admin/admin-server/src/main/java/cc/riskswap/trader/admin/controller/ExchangeController.java)
- Service：[ExchangeService.java](../../trader-admin/admin-server/src/main/java/cc/riskswap/trader/admin/service/ExchangeService.java)
- Dao：ExchangeDao（pageQuery）

接口：
- GET /exchange/list（ExchangeListQuery）
  - Response：ResData<PageDto<ExchangeDto>>

### 交易日历

- Controller：[CalendarController.java](../../trader-admin/admin-server/src/main/java/cc/riskswap/trader/admin/controller/CalendarController.java)
- Service：[CalendarService.java](../../trader-admin/admin-server/src/main/java/cc/riskswap/trader/admin/service/CalendarService.java)
- Dao：[CalendarDao.java](../../trader-admin/admin-server/src/main/java/cc/riskswap/trader/admin/dao/CalendarDao.java)

接口：
- GET /calendar/list（CalendarListQuery）
  - 过滤条件：exchange、startDate、endDate
  - 排序：date 倒序
  - Response：ResData<PageDto<CalendarDto>>

内部能力（当前未暴露为 Controller 接口）：
- CalendarService.getCalendarCN(year)：返回指定年份的 date/is_open 二维结果集
  - 数据来源：CalendarDao.list(year)（当前实现返回 null，属于待补齐能力）

## 数据模型（MySQL）

表定义参考：[db.sql](../../trader-admin/admin-server/src/main/resources/database/db.sql)

### exchange

- code（PK）
- name
- timezone
- created_at、updated_at

### calendar

- exchange + date（联合主键）
- open（0/1，是否开市）
- pre_date（前一交易日）
- created_at、updated_at

## 关键流程（时序）

### 查询交易日历

1. 前端输入交易所 code 与日期范围
2. GET /calendar/list
3. CalendarService 组合条件查询 calendar 表并分页返回
4. 前端渲染表格与分页

## 约束与扩展点

- CalendarDao.list(year) 当前未实现（返回 null），导致 getCalendarCN(year) 无法基于数据库补齐节假日修正数据；如要使用该能力需完善 DAO 实现与对应 Controller 暴露。
